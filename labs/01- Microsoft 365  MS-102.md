# Microsoft 365 / MS-102 Lab

## Overview

This lab demonstrates hands-on administration of Microsoft 365 services
using real Microsoft admin portals. The lab focuses on identity management,
email administration, and collaboration tools commonly handled by
IT Support and Microsoft 365 Administrators.

---

## Environment

- Microsoft 365 Tenant (HomeLab)
- Microsoft Entra ID (Azure AD)
- Exchange Online
- Microsoft Teams
- Windows 10/11 client device

---

## Tasks Completed

### Microsoft Entra ID (Identity Management)

- Accessed Microsoft Entra admin center
- Reviewed tenant configuration
- Created cloud users
- Assigned Microsoft 365 licenses
- Verified user account status and sign-in readiness

### Exchange Online (Email Administration)

- Accessed Exchange admin center
- Created a shared mailbox (Support)
- Configured shared mailbox permissions:
  - Full Access
  - Send As
- Verified mailbox delegation and access

### Microsoft Teams (Collaboration)

- Created a Microsoft Team
- Created a channel within the Team
- Added users to the Team
- Verified member access and collaboration readiness
